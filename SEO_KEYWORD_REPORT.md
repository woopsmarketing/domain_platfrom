## 중요: Google은 `<meta keywords>`를 무시합니다

SEO에 실제 영향을 주는 요소 (중요도 순):
1. **title 태그** — 검색 결과에 직접 표시
2. **본문 콘텐츠** — 페이지 내 실제 텍스트 (h1/h2/h3/p)
3. **meta description** — 클릭률(CTR) 영향
4. **내부 링크 앵커 텍스트** — 페이지 간 연관성 신호
5. ~~meta keywords~~ — Google 무시 (효과 없음)
