## 0. 한 장 요약 (Executive Summary)

| 항목 | 내용 |
|---|---|
| **제품** | 무료 도메인 분석 도구 (만료/경매 도메인 DB + DA/DR/TF + 12종 유틸 툴) |
| **타겟** | 한국 SEO 대행사 실무자 + SEO 관심 블로거/마케터 (B2B 실무자 중심) |
| **1차 목표** | 오가닉 월 방문자 0 → **8,000 (12주 후)**, 도메인 검색 실행 **월 2,000건** |
| **2차 목표** | **프리미엄 도메인 마켓플레이스 구매 신청 전환** (현재 재고 102개, x1.5 마진) |
| **수익 모델** | 모든 도구 완전 무료 → 신뢰 형성 → `/marketplace` 프리미엄 도메인 리셀 판매 (이메일+텔레그램 구매 신청) |
| **포지셔닝** | "한국에서 유일한, 만료 도메인부터 DA까지 **한 곳에서 무료로** 확인하는 도구" |
| **월 예산** | 구글 광고 30만원 (일 1만원) + 오가닉(블로그 자동화) |
| **핵심 KPI** | ① 오가닉 세션 ② 도메인 검색 이벤트 ③ 마켓플레이스 CTA 클릭률 ④ 광고 CTR ≥5% ⑤ 광고 전환당 비용(CPA) ≤ 3,000원 ⑥ 마켓플레이스 구매 신청 수 |

**한 줄 전략**: 광고는 "**구매 의도 키워드**"에만 좁게 태우고(예산 보호), 오가닉 블로그로 "**정보 탐색 키워드**"를 전량 흡수해서 툴 페이지로 내부링크 순환시킨다.

### 2.1 1차 페르소나: "SEO 실무자 민수 (32세)"

| 항목 | 값 |
|---|---|
| 직업 | SEO 대행사 주니어~미들 (1~4년차) |
| 업무 | 클라이언트 백링크 구축, PBN 도메인 소싱, 경쟁사 분석 |
| 페인포인트 | Ahrefs 유료 구독 아까움. 한국어 도메인 DB가 없어서 영어 사이트 뒤짐. 여러 툴을 탭 5개 열어야 함 |
| 검색어 습관 | "만료 도메인 찾는 법", "DA 높은 도메인", "aged domain 한국", "백링크 체크" |
| 가치 제안 | 탭 하나로 전부 해결. 한국어. 무료. |
| 전환 경로 | 검색 → 블로그 → 툴 페이지 → 북마크/재방문 → (Pro) |

### 2.3 고객 여정 맵 (프리미엄 도메인 마켓 깔때기)

```
[인지]         [신뢰 형성]       [쇼룸 인지]          [구매 의사]         [전환]
블로그 정보글 → 툴 12종 무료 → /market-history → /marketplace    → 구매 신청
구글 광고   →  사용 (재방문) → (실제 거래가 학습) (102개 검증)    → 이메일+텔레그램
커뮤니티    →  북마크        → 만료 도메인 DB      엄선 도메인 열람
                             (시장 감각 훈련)
```

**핵심 깔때기 원칙**: 광고 클릭 → 바로 마켓플레이스로 보내지 **않는다**. 도구 사용으로 신뢰를 먼저 쌓고, market-history 데이터로 "이런 도메인이 실제 거래되는구나"를 학습시킨 후에 `/marketplace`의 "엄선된 102개"가 그제서야 가치 있게 보인다. 이게 **Ahrefs 모델**(무료 백링크 체커 → 유료 구독)과 동일한 구조.

**병목 지점**: "탐색 → 고려" 구간. 도구만 쓰고 마켓플레이스 존재를 모르는 유저.
→ 해결: ① **홈 히어로 하단 마켓플레이스 CTA 배너** (구현 완료) ② **`/market-history` 상단 CTA 배너** (구현 완료) ③ 블로그 포스트 하단 "관련 추천 도메인 3개" 위젯 (추후) ④ 결과 페이지 SNS 공유 버튼 (추후)

## 4. Google Ads 캠페인 설계

### 4.1 캠페인 구조

```
캠페인 A: 만료/경매 도메인 (일 6,000원, 월 18만원) ← 핵심
 ├─ 광고그룹 A1: 만료 도메인
 ├─ 광고그룹 A2: 경매 도메인
 └─ 광고그룹 A3: PBN / aged domain

캠페인 B: DA / 백링크 체커 (일 3,000원, 월 9만원)
 ├─ 광고그룹 B1: DA 체커
 └─ 광고그룹 B2: 백링크 확인

캠페인 C: 브랜드 보호 (일 500원, 월 1.5만원)
 └─ 광고그룹 C1: 도메인체커 브랜드명

캠페인 D: 리마케팅 (일 500원, 월 1.5만원) [4주차 이후 활성화]
 └─ 방문자 대상 디스플레이
```

**합계: 일 10,000원 (월 30만원)**

### 4.2 입찰 전략

| 단계 | 기간 | 전략 | 이유 |
|---|---|---|---|
| 1단계 | 1~2주차 | **클릭 수 최대화** (CPC 상한 500원) | 데이터 수집 |
| 2단계 | 3~4주차 | **전환 수 최대화** | GA4 전환 이벤트 학습 |
| 3단계 | 5주차~ | **타겟 CPA** (목표 2,500원) | 최적화 |

**중요**: 1단계에서 반드시 GA4 "도메인 검색 실행" 이벤트를 전환으로 가져오기 설정 (4.5절 참고).
