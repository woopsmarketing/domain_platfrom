## 2. 기존 캠페인 완전 중지

**경로**: Google Ads → 좌측 메뉴 → **캠페인**

1. 전체 기간 필터를 **"이번 달"** 로 설정
2. "상태"가 **사용 설정** 또는 **제한됨**인 기존 캠페인 **모두 체크박스 선택**
3. 상단 액션 바 **일시중지** 클릭
4. 확인 팝업에서 **일시중지**
5. 필터를 "모든 캠페인"으로 바꿔서 **일시중지 상태**로 표시되는지 검증

> ⚠️ **삭제하지 말 것**. 일시중지만. 과거 데이터는 학습/대조용으로 남긴다.

## 3. 전환 액션 설정 (GA4 → Google Ads Import)

> ✅ **GA4 주요 이벤트 등록은 이미 완료됨** (2026-04-15 수동 등록).
> 이 섹션은 **GA4에 등록된 주요 이벤트를 Google Ads 전환 액션으로 가져오는** 단계.

**경로**: Google Ads → 도구 (🔧 아이콘) → **측정** → **전환**

### 3-1. GA4에서 전환 가져오기

1. 상단 **+ 새 전환 액션** → **가져오기** → **Google 애널리틱스 4 속성** → **웹**
2. GA4 속성 `G-9FP3RGJNZV` 선택
3. 가져올 이벤트 체크 (GA4에서 이미 주요 이벤트로 표시돼 있음):
   - ☑ `domain_search_submit`
   - ☑ `tool_used`
   - ☑ `marketplace_click`
   - ☑ `purchase_request`
4. **가져오기 및 계속** 클릭

> ⚠️ **만약 4개 이벤트가 목록에 안 보이면**: GA4가 Google Ads에 이벤트를 동기화하는 데 최대 24시간 걸릴 수 있음. 오늘 등록했으면 내일 아침에는 나타남. 안 보이면 하루 기다렸다가 재시도.

### 3-2. 전환 액션별 설정

가져온 각 전환을 클릭하고 아래 값으로 설정.

| 전환 이름 | 카테고리 | 가치 | 카운트 방법 | 클릭 기여 기간 | 목표 유형 |
|---|---|---|---|---|---|
| `domain_search_submit` | **리드** → 양식 제출 | 동일 값 **500원** | **1회** (전환당) | 30일 | 보조 |
| `tool_used` | **페이지뷰** → 페이지 조회 | 동일 값 **300원** | **1회** | 30일 | 보조 |
| `marketplace_click` | **리드** → 아웃바운드 클릭 | 동일 값 **3,000원** | **1회** | 30일 | **기본** |
| `purchase_request` | **구매** → 구매 | 동일 값 **50,000원** | **1회** | 60일 | **기본** |

> 가치(₩)는 가상 프록시 값. 실제 매출이 아니라 **입찰 알고리즘이 전환 우선순위를 매기는 용도**다. 3,000 vs 50,000 비율이 중요하지 절대값은 중요하지 않다.

### 3-3. 기본 목표 지정

1. 우측 **기본 목표로 사용** 설정
2. `marketplace_click`과 `purchase_request`만 **기본**
3. `domain_search_submit`과 `tool_used`는 **보조** (분석용)

> 이렇게 하면 Smart Bidding이 학습할 때 **마켓플레이스 클릭/구매**를 최우선으로 본다. 나머지는 엔게이지먼트 지표로만 참고.

## 14. Google Ads 정책 주의사항 (중요)

### 14.1 절대 해서는 안 될 것

1. **광고 카피에 도박/토토/카지노/베팅 단어 직접 사용 금지**
   → 즉시 계정 정지 위험
2. **성인·19금 단어 직접 사용 금지**
3. **불법 스트리밍·토렌트·해킹 단어 금지**
4. **"보장", "100%", "절대"** 같은 과장 표현 금지 → 자동 검수 탈락
5. **대문자만 사용한 헤드라인**(영어) 금지

### 14.2 캠페인 C (블랙키워드 상위노출) 특별 주의

- **"블랙키워드" 자체는 한국 SEO 업계 표준 슬랭**이라 자동 검수에서 거부될 가능성은 낮다. 다만 구글 **수동 검수**에서 사람 검수자가 맥락을 이해 못 하고 거부할 가능성은 존재 → 거부되면 해당 광고/키워드만 일시중지 후 재작성.
- **사용해도 되는 표현**: "블랙키워드", "블랙 SEO", "상위노출", "니치 도메인", "고수익", "틈새", "업종 특화"
- **절대 사용 금지**: "토토", "카지노", "바카라", "성인", "19금", "야동", "중계", "도박", "베팅" → 직접 명사 사용 시 즉시 거부
- **랜딩 URL**: `/marketplace` (중립) 유지. 블랙 전용 랜딩 페이지 만들지 않음
- 1주차 운영 후 **일부 키워드 "제한됨"** 표시 가능 → 해당 키워드만 즉시 일시중지, 다른 키워드는 유지
- **정책 경고 메일 오면 즉시 카피 검토** → 거부된 광고만 제거하고 재작성
- 계정 단위 경고 2회 누적 시 → 캠페인 C 전체 일시중지 후 재설계

### 14.3 Google Ads 자동 검수 통과 팁

- 구체적 숫자 사용 ("102개", "3초", "12가지")
- 동사는 강하지만 과장 없는 것 ("확인하세요", "분석하세요" O / "돈 버세요" X)
- 기호 사용은 `·` `✔` `→` 만 사용 (이모지 최소화)
- 구두점 2개 이상 연속 금지 (`!!`, `??`, `...`)

### 15.1 헤드라인 구조 공식

```
[특징] + [숫자/구체성] + [감정 훅]
"DA 50+ 도메인 지금 확인"
 ^특징   ^구체성       ^긴급성
```

### 15.2 헤드라인별 심리 원칙 매핑 (샘플)

| 헤드라인 | 주 원칙 | 보조 원칙 |
|---|---|---|
| "만료 도메인 DB 무료 공개" | 상호성(무료) | 권위(DB) |
| "회원가입 없이 즉시 검색" | 저항 제거 | 즉시성 |
| "Ahrefs 대신 무료로 분석" | 손실 회피(비용) | 앵커링 |
| "프리미엄 도메인 102개" | 희소성 | 구체성 |
| "놓치면 경쟁사가 가져갑니다" | 손실 회피 | 긴급성 |
| "한국 SEO 대행사가 쓰는 툴" | 사회적 증명 | 호감(자기 정체성) |
| "버려진 가치, 당신의 기회" | 감정 훅 | 서사 |
| "DA 50+ 도메인 지금 확인" | 구체성 | 즉시성 |
| "SEO 실무자의 필수 도구" | 권위 | 정체성 강화 |
| "중복 판매 절대 없음" | 희소성 | 신뢰 |

### 15.3 설명문 구조 공식

```
[1 문장: 가치 제안] + [2 문장: 차별점] + [3 문장: 행동 촉구]
```

예시:
> "도메인 백링크를 무료로 확인. [가치] Ahrefs 대체, 한국어 전용, 회원가입 불필요. [차별] 경쟁사 분석까지 한 번에. [CTA]"

### 15.4 카피에서 쓰지 않은 것 (의도적)

- **"최고"**, **"1위"** 과장 X → 광고 정책 리스크 + 불신
- **"지금 즉시"** 상투어 X → 마케팅 냄새 강함
- **이모지 폭탄** X → 프로페셔널 톤 유지
- **느낌표 남발** X → 감정 과잉 신호
